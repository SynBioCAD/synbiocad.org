--- layout: default title: SynBioCAD | An open-source visualization framework for synthetic biology ---

SynBioCAD is an open-source visualization framework for visualizing synthetic biology designs built upon the Synthetic Biology Open Language (SBOL).

You can use SynBioCAD:

Stand-alone in the browser as an interactive genetic circuit design tool

The user interface is still very much in pre-alpha stage, so please help by reporting any issues to the GitHub issue tracker.

As a REST API to add SynBioCAD rendering to a separate website or tool

As a JavaScript/TypeScript library to embed SynBioCAD rendering directly into an existing Web application
